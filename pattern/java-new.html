Java 8 - Java 23 Arasında Gelen ve En Çok Kullanılan Özellikler

Java 8 Özellikleri

1. Lambda İfadeleri
Java 8 ile gelen en önemli yeniliklerden biri lambda ifadeleridir. Fonksiyonel programlamayı destekler ve kodun okunabilirliğini artırır. 

List<String> names = Arrays.asList("Ali", "Ayşe", "Mehmet");
names.forEach(name -> System.out.println(name));
Detaylar: Burada forEach metodu ile listedeki her eleman için name -> System.out.println(name) ifadesi çalıştırılır. Lambda sayesinde anonim fonksiyon tanımlanmış olur.

2. Stream API
Büyük veri kümelerini filtrelemek, sıralamak veya dönüştürmek için kullanılır. 

List<String> names = Arrays.asList("Ali", "Ayşe", "Mehmet", "Ahmet");
List<String> filteredNames = names.stream()
                                     .filter(name -> name.startsWith("A"))
                                     .collect(Collectors.toList());
System.out.println(filteredNames); // Çıktı: [Ali, Ayşe, Ahmet]
Detaylar: filter ile "A" harfiyle başlayan isimler filtrelenir ve sonuç yeni bir liste olarak toplanır.

3. Optional Sınıfı
NullPointerException hatalarını önlemek amacıyla kullanılır. 

Optional<String> name = Optional.ofNullable("Ali");
name.ifPresent(System.out::println);
Detaylar: ofNullable metodu ile Optional bir değer oluşturulur. ifPresent, değeri null değilse işlemi gerçekleştirir.

Java 9 Özellikleri

1. Modül Sistemi
Kodları daha iyi organize etmek için kullanılır. Uygulamaların bağımlılıklarını tanımlamak için gereklidir. 

module-info.java
module com.example.app {
    requires java.logging;
}
Detaylar: Modüler yapı, büyük projelerde bağımlılıkları yönetmeyi kolaylaştırır.

2. Stream API Geliştirmeleri
Yeni metotlar eklendi: takeWhile, dropWhile, ofNullable. 

List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
numbers.stream()
       .takeWhile(n -> n < 4)
       .forEach(System.out::println);
Detaylar: takeWhile, belirtilen koşul doğru olduğu sürece öğeleri alır. İlk yanlış olduğunda işlemi sonlandırır.

Java 10 Özellikleri

1. var Anahtar Kelimesi
Değişken türlerini otomatik olarak belirler. 

var message = "Merhaba Dünya";
System.out.println(message);
Detaylar: var, veri türünü otomatik olarak belirler, ama sadece lokal değişkenler için kullanılır.

Java 11 Özellikleri

1. String API Geliştirmeleri
isBlank, lines, strip, repeat gibi yeni metotlar eklenmiştir. 

String text = " Java ";
System.out.println(text.strip()); // "Java"
System.out.println("Hello".repeat(3)); // "HelloHelloHello"
Detaylar: strip boşlukları kaldırırken, repeat belirtilen sayıda tekrarlama yapar.

Java 14 Özellikleri

1. Switch İfadeleri
Daha okunabilir bir switch yapısı sağlar. 

int day = 3;
String dayName = switch (day) {
    case 1 -> "Pazartesi";
    case 2 -> "Salı";
    case 3 -> "Çarşamba";
    default -> "Bilinmiyor";
};
System.out.println(dayName);
Detaylar: Lambda yapısıyla uyumlu ve daha az kodla çalışır.

Java 17 Özellikleri

1. Pattern Matching for instanceof
instanceof kullanarak doğrudan dönüşüm yapılır. 

Object obj = "Merhaba";
if (obj instanceof String s) {
    System.out.println(s.toUpperCase());
}
Detaylar: s değişkeni instanceof ile otomatik olarak tanımlanır.

Java 21 Özellikleri

1. Record Pattern Matching
record yapılarıyla veri modellemede kolaylık sağlar. 

record Point(int x, int y) {}

static void printPoint(Object obj) {
    if (obj instanceof Point(int x, int y)) {
        System.out.println("X: " + x + ", Y: " + y);
    }
}
Detaylar: Point bir record yapısıdır ve bu yapı ile doğrudan veriler işlenebilir.

2. String Templates (Önizleme)
String birleştirmelerini daha kolay hale getirir. 

String name = "Ali";
String message = STR."Merhaba, \{name}!";
Detaylar: Bu özellik sayesinde, \{} ile değişkenler doğrudan metin içerisine gömülür.
